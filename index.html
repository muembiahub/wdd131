About Me

WDD 131 – Dynamic Web Fundamentals is an introductory course in web development that covers the basics of HTML, CSS, and JavaScript. Students will learn how to create interactive and user-friendly websites using these technologies. The course will focus on building a strong foundation in web development and preparing students for more advanced courses in the field. By the end of the course, students will have the skills and knowledge to create their own websites and web applications.

Course Objectives

  • Understand the basics of HTML, CSS, and JavaScript
  • Create interactive and user-friendly websites
  • Build a strong foundation in web development
  • Prepare for more advanced courses in web development

Course Schedule

  • Week 1: Introduction to HTML
  • Week 2: Styling with CSS
  • Week 3: JavaScript Basics
  • Week 4: Advanced JavaScript
  • Week 5: Project Week
